# 权限系统设计

## 📌 核心概念
权限系统通常包含身份认证、资源定义、角色设计、权限点映射和前后端协同校验。常见模型有 RBAC、ABAC。

## 💻 代码示例
```js
const can = (user, permission) => user.permissions.includes(permission)
if (can(user, 'order:edit')) {
  showEditButton()
}
```

## 🛠 实战场景
中后台系统里菜单、按钮、路由、接口权限往往需要统一设计。

## 🎯 面试怎么问
- 前端权限控制和后端权限控制边界在哪？
- RBAC 的核心结构是什么？
- 动态路由如何配合权限系统？

## ⚠️ 易错点 / 高阶拓展
前端权限只能做体验控制，真正安全边界必须落在服务端。
