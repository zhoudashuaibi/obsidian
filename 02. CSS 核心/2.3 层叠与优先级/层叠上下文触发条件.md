# 层叠上下文触发条件

## 📌 核心概念
层叠上下文是元素在 z 轴上的独立绘制环境。创建后，内部元素的 `z-index` 只在该上下文内比较。常见触发条件包括根元素、定位元素配合 `z-index`、`opacity < 1`、`transform`、`filter` 等。

## 💻 代码示例
```css
.modal {
  position: relative;
  z-index: 10;
}

.card {
  transform: translateZ(0);
}

.mask {
  position: fixed;
  z-index: 999;
}
```

## 🛠 实战场景
弹窗被遮住、悬浮层层级错乱、`z-index` 怎么调都不生效，很多问题本质不是数字大小，而是进入了不同层叠上下文。

## 🎯 面试怎么问
- 什么是层叠上下文？
- 哪些 CSS 属性会创建层叠上下文？
- 为什么子元素 `z-index: 9999` 还是盖不过外部元素？

## ⚠️ 易错点 / 高阶拓展
回答时要区分“层叠顺序”和“层叠上下文”。高阶常追问 stacking order 细节，以及 `isolation`、`contain` 等新属性影响。
