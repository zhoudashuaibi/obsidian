# 防抖与节流原理与实现

## 📌 核心概念
防抖是在高频触发结束后一段时间再执行，节流则是在固定时间窗口内最多执行一次。它们都用于限制事件处理频率，但适合的交互场景不同。

## 💻 代码示例
```js
function debounce(fn, delay) {
  let timer = null;
  return (...args) => {
    clearTimeout(timer);
    timer = setTimeout(() => fn(...args), delay);
  };
}

function throttle(fn, delay) {
  let last = 0;
  return (...args) => {
    const now = Date.now();
    if (now - last >= delay) {
      last = now;
      fn(...args);
    }
  };
}
```

## 🛠 实战场景
输入搜索、窗口 resize 常用防抖；滚动监听、拖拽位置同步、按钮限频更适合节流。
