# 色彩对比度与屏幕阅读器适配

## 📌 核心概念
可访问性不仅是标签语义，还包括视觉可辨识度和读屏体验。足够的色彩对比度能帮助低视力用户看清内容，而屏幕阅读器适配要求文本顺序、标签语义和可读描述合理。

## 💻 代码示例
```html
<label for="email">邮箱</label>
<input id="email" type="email" aria-describedby="email-tip">
<p id="email-tip">请输入常用邮箱地址</p>
```

## 🛠 实战场景
表单、按钮、错误提示、图表说明都要考虑颜色不能成为唯一信息来源，同时读屏用户也应能获得等价信息。

## 🎯 面试怎么问
- 为什么颜色不能作为唯一提示手段？
- 色彩对比度大概应满足什么标准？
- 图片和图标要不要补文本说明？

## ⚠️ 易错点 / 高阶拓展
很多无障碍问题肉眼不容易发现，最好结合浏览器审计工具和实际读屏测试，而不是只看设计稿。
