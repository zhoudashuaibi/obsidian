# SEO 关键 meta 标签

## 📌 核心概念
`title`、`description`、`robots`、`viewport` 等 meta 信息会影响搜索结果展示、抓取策略和移动端呈现。它们本身不保证排名，但能帮助搜索引擎更准确理解页面。

## 💻 代码示例
```html
<title>前端面试知识库</title>
<meta name="description" content="系统整理前端面试知识体系">
<meta name="robots" content="index,follow">
<meta name="viewport" content="width=device-width, initial-scale=1">
```

## 🛠 实战场景
内容站、官网、落地页都需要根据页面内容动态设置 title 和 description，否则搜索展示效果和点击率会明显受影响。

## 🎯 面试怎么问
- 哪些 meta 标签和 SEO 最相关？
- meta keywords 现在还有用吗？
- 为什么 title 和 description 要做页面级配置？

## ⚠️ 易错点 / 高阶拓展
SEO 不是只靠 meta 标签。真正影响更大的还有内容质量、站点结构、可抓取性、性能和外链。
