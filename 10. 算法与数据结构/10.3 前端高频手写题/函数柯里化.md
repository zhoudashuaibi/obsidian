# 函数柯里化

## 📌 核心概念
柯里化是把接收多个参数的函数拆成一系列接收单个或部分参数的函数，便于参数复用和函数组合。

## 💻 代码示例
```js
function curry(fn, ...args) {
  return (...rest) => {
const allArgs = [...args, ...rest]
return allArgs.length >= fn.length ? fn(...allArgs) : curry(fn, ...allArgs)
  }
}
```

## 🛠 实战场景
埋点函数预置公共参数、校验器组合、函数式编程库都大量使用柯里化。

## 🎯 面试怎么问
- 柯里化的价值是什么？
- fn.length 在这里起什么作用？
- 柯里化和偏函数有何区别？

## ⚠️ 易错点 / 高阶拓展
不是所有函数都适合柯里化，过度使用会降低可读性。
