# 深拷贝实现

## 📌 核心概念
深拷贝需要递归复制引用类型，并处理数组、对象、Date、RegExp、循环引用等情况。JSON 序列化只能覆盖最简单场景。

## 💻 代码示例
```js
function deepClone(value, cache = new WeakMap()) {
  if (typeof value !== 'object' || value === null) return value
  if (cache.has(value)) return cache.get(value)
  const result = Array.isArray(value) ? [] : {}
  cache.set(value, result)
  for (const key in value) result[key] = deepClone(value[key], cache)
  return result
}
```

## 🛠 实战场景
这题常拿来考察递归、类型判断、WeakMap 和边界处理。

## 🎯 面试怎么问
- JSON.parse(JSON.stringify()) 的问题有哪些？
- 为什么要用 WeakMap？
- 如何复制 Date、Map、Set？

## ⚠️ 易错点 / 高阶拓展
真正的生产深拷贝要考虑属性描述符、原型链、不可枚举属性等，面试手写版通常不必无限上纲。
