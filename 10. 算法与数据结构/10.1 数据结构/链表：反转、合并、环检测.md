# 链表：反转、合并、环检测

## 📌 核心概念
链表题重点在指针操作。反转链表考察指针翻转，合并链表考察有序比较，环检测通常用快慢指针。

## 💻 代码示例
```js
function reverseList(head) {
  let prev = null
  let cur = head
  while (cur) {
const next = cur.next
cur.next = prev
prev = cur
cur = next
  }
  return prev
}
```

## 🛠 实战场景
前端虽然很少直接写链表业务，但这是面试里考察指针思维和边界处理的经典题。

## 🎯 面试怎么问
- 如何原地反转链表？
- 合并两个有序链表怎么写？
- 为什么快慢指针能判断有环？

## ⚠️ 易错点 / 高阶拓展
链表题最容易丢节点或造成死循环，写之前先画指针变化图。
