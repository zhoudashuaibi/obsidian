# 哈希表：冲突处理

## 📌 核心概念
哈希表通过哈希函数把 key 映射到数组下标，实现近似 O(1) 查找。冲突不可避免，常见处理有拉链法、开放寻址法。

## 💻 代码示例
```js
const map = new Map()
map.set('name', 'zhou')
console.log(map.get('name'))
```

## 🛠 实战场景
缓存、去重、计数器、索引映射，几乎都是哈希思想的直接应用。

## 🎯 面试怎么问
- 为什么哈希冲突一定会发生？
- 拉链法和开放寻址法差异？
- JS 的 Map 和普通对象有何不同？

## ⚠️ 易错点 / 高阶拓展
哈希表平均复杂度高效，不代表最坏情况也一定快；哈希函数质量和扩容策略都很关键。
