# 堆：优先队列应用

## 📌 核心概念
堆是完全二叉树结构，常用数组实现。最大堆/最小堆支持快速取最值，是优先队列的经典底层结构。

## 💻 代码示例
```js
class MinHeap {
  constructor() { this.data = [] }
  peek() { return this.data[0] }
}
```

## 🛠 实战场景
定时任务调度、TopK、合并 K 个有序链表、优先级任务系统都常用堆。

## 🎯 面试怎么问
- 堆和排序数组的区别？
- 为什么取堆顶是 O(1)？
- 前端业务里哪里会用到优先队列？

## ⚠️ 易错点 / 高阶拓展
堆不是完全有序结构，只保证父子节点局部有序。
