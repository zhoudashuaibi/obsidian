# 快速排序与归并排序手写

## 📌 核心概念
快排采用分治 + 基准划分，平均 O(n log n)；归并排序通过拆分后合并，时间稳定 O(n log n)，但额外空间更高。

## 💻 代码示例
```js
function mergeSort(arr) {
  if (arr.length <= 1) return arr
  const mid = arr.length >> 1
  return merge(mergeSort(arr.slice(0, mid)), mergeSort(arr.slice(mid)))
}
```

## 🛠 实战场景
排序题常被用来考察递归、分治、稳定性和时间复杂度分析能力。

## 🎯 面试怎么问
- 快排最坏复杂度为什么退化到 O(n^2)？
- 归并排序为什么稳定？
- 实际工程里排序选择要看什么？

## ⚠️ 易错点 / 高阶拓展
不要只会背复杂度，最好能说出快排的原地优势与归并的稳定性取舍。
