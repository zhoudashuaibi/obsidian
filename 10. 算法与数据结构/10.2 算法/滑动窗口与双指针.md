# 滑动窗口与双指针

## 📌 核心概念
滑动窗口和双指针适合处理连续区间、子串、数组有序区间等问题，核心是维护左右边界并利用题目单调性减少重复计算。

## 💻 代码示例
```js
function maxSum(nums, k) {
  let sum = 0
  for (let i = 0; i < k; i++) sum += nums[i]
  let ans = sum
  for (let r = k; r < nums.length; r++) {
sum += nums[r] - nums[r - k]
ans = Math.max(ans, sum)
  }
  return ans
}
```

## 🛠 实战场景
最长无重复子串、最短覆盖子串、盛最多水的容器、删除有序数组重复项都属于高频题。

## 🎯 面试怎么问
- 什么时候用滑动窗口，什么时候用双指针？
- 窗口内状态如何维护？
- 为什么很多题能从 O(n^2) 降到 O(n)？

## ⚠️ 易错点 / 高阶拓展
窗口题的关键不只是移动指针，而是想清楚窗口不变量。
