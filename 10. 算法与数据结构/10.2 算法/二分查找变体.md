# 二分查找变体

## 📌 核心概念
二分查找不只用于找某个值，还常用于找左边界、右边界、最小满足条件值等“答案单调”问题。

## 💻 代码示例
```js
function lowerBound(nums, target) {
  let l = 0, r = nums.length
  while (l < r) {
const mid = (l + r) >> 1
if (nums[mid] < target) l = mid + 1
else r = mid
  }
  return l
}
```

## 🛠 实战场景
搜索插入位置、查找第一个等于目标值、开方、容量最小化等题都能抽象成二分。

## 🎯 面试怎么问
- 为什么很多题是“二分答案”而不是二分数组”？
- 左闭右开和闭区间写法有什么区别？
- 如何避免死循环？

## ⚠️ 易错点 / 高阶拓展
二分的核心不是背模板，而是先确认单调性，再统一边界定义。
