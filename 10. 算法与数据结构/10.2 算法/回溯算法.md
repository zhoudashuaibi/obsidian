# 回溯算法

## 📌 核心概念
回溯本质是带状态恢复的深度优先搜索，适合组合、排列、子集、路径搜索这类需要穷举解空间的问题。

## 💻 代码示例
```js
function backtrack(path, nums, used, ans) {
  if (path.length === nums.length) {
ans.push(path.slice())
return
  }
  for (let i = 0; i < nums.length; i++) {
if (used[i]) continue
used[i] = true
path.push(nums[i])
backtrack(path, nums, used, ans)
path.pop()
used[i] = false
  }
}
```

## 🛠 实战场景
全排列、N 皇后、组合总和、括号生成都是回溯模板题。
