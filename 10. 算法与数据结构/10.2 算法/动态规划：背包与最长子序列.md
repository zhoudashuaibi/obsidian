# 动态规划：背包与最长子序列

## 📌 核心概念
动态规划适合解决具有重叠子问题和最优子结构的问题。背包和最长子序列是最经典两类模型，关键是定义状态和转移方程。

## 💻 代码示例
```js
function lengthOfLIS(nums) {
  const dp = Array(nums.length).fill(1)
  for (let i = 0; i < nums.length; i++) {
for (let j = 0; j < i; j++) {
  if (nums[i] > nums[j]) dp[i] = Math.max(dp[i], dp[j] + 1)
}
  }
  return Math.max(...dp)
}
```
