# 合成线程与光栅化

## 📌 核心概念
现代浏览器会把页面拆成图层，由合成线程调度图层合成；光栅化则是把图层内容转换为实际像素位图。滚动、动画性能好不好，很大程度取决于是否能在合成阶段完成。

## 💻 代码示例
```text
transform / opacity -> 更容易只走合成
width / top -> 往往触发布局或绘制
```

## 🛠 实战场景
做复杂动画、长列表滚动、吸顶浮层时，理解合成线程和光栅化有助于解释为什么某些动画不卡，而某些会掉帧。

## 🎯 面试怎么问
- 什么是合成线程？
- 光栅化在渲染流程里做什么？
- 为什么 transform 动画通常更流畅？

## ⚠️ 易错点 / 高阶拓展
合成层不是越多越好。过多图层会带来显存占用、管理开销和栅格化压力。
